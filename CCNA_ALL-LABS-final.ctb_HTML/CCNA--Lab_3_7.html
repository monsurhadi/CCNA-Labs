Lab 3


Lab No : 03 (Default Routing)

Steps:

  1. First get all routers interface up.
  2. Assign IPs to the PCs.
  3. Then default route IP table.
  4. Ping all PC IPs to check connectivity.

Router0:

Router(config)#ip route 0.0.0.0 0.0.0.0 10.0.0.2 // (Dynamic Routing)

Ctrl + z

Router#show ip route

Codes: L - local, C - connected, S - static, R - RIP, M - mobile, B - BGP

D - EIGRP, EX - EIGRP external, O - OSPF, IA - OSPF inter area

N1 - OSPF NSSA external type 1, N2 - OSPF NSSA external type 2

E1 - OSPF external type 1, E2 - OSPF external type 2, E - EGP

i - IS-IS, L1 - IS-IS level-1, L2 - IS-IS level-2, ia - IS-IS inter area

* - candidate default, U - per-user static route, o - ODR

P - periodic downloaded static route

Gateway of last resort is 10.0.0.2 to network 0.0.0.0
