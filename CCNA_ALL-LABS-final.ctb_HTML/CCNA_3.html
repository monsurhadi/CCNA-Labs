CCNA Hands-On Labs Collection

About This Repository

This repository contains my complete CCNA hands-on lab collection built of practical learning, troubleshooting, and network simulation using Cisco Packet Tracer.

The goal of this project was not simply to complete CCNA topics, but to gain real-world networking experience by building networks from scratch, breaking configurations,

identifying issues, and troubleshooting them until they worked correctly.

This repository documents that journey.

Labs Covered

  • Interface Configuration
  • Static Routing
  • Default Routing
  • EIGRP
  • RIP v2
  • OSPF

Network Services

  • DHCP
  • Syslog Server
  • NAT (Network Address Translation)

Switching

  • VLAN Configuration
  • Inter-VLAN Routing (Router-on-a-Stick)
  • VTP (VLAN Trunking Protocol)
  • STP (Spanning Tree Protocol)
  • RSTP (Rapid Spanning Tree Protocol)
  • EtherChannel (PAgP/LACP)

Security

  • Standard ACLs
  • Extended ACLs

WAN Technologies

  • PPP Authentication
  • PAP Authentication
  • CHAP Authentication

High Availability

  • HSRP (Hot Standby Router Protocol)
